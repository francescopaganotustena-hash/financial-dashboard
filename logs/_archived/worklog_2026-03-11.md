## Come ripartire rapidamente
1. Avvio backend locale:
   `cd backend && uvicorn app.main:app --host 0.0.0.0 --port 8000`
2. Avvio frontend locale:
   `cd frontend && VITE_API_URL=http://10.0.0.103:8000 npm run dev -- --host 0.0.0.0 --port 3001`
3. Check rapidi:
   - `npm run lint`
   - `npx tsc --noEmit -p tsconfig.app.json`
   - `npm run build`
